# 문제
* [백준 수들의 합 5 링크](https://www.acmicpc.net/problem/2018)

# 내용
* 연속된 수의 합을 구하는 문제입니다. 투포인터로 해결했습니다.

# 풀이
```java
package boj_2018;

import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int answer = 0;

        int start = 1;
        int end = 1;
        int sum = 1;

        while (start <= end) {
            if (sum == n) answer++;
            if (sum < n) {
                sum += ++end;
            } else {
                sum -= start++;
            }
        }
        System.out.println(answer);
    }

}




```
